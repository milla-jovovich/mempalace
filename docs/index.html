Quick start

Four commands to a searchable memory.

Install, initialize, mine, search. Everything stays on your machine — no API key, no cloud, no subscription.

setup — bash 
# Install the package
$ pip install mempalace

# Set up your world — team, projects, preferences
$ mempalace init ~/projects/myapp

# Mine your data (code, docs, chats — all of it)
$ mempalace mine ~/projects/myapp
$ mempalace mine ~/chats/ --mode convos
$ mempalace mine ~/chats/ --mode convos --extract general

# Search anything you’ve ever discussed
$ mempalace search "why did we switch to GraphQL"
→ 12 results · verbatim · 47ms

Offline LLMs

Llama, Mistral, local models

Local models don’t speak MCP yet. Two patterns: load your ~170-token wake-up into context, or query verbatim on demand.

Entire memory stack runs offline — zero cloud calls.

# Load entire world (~170 tokens)
$ mempalace wake-up > ctx.txt

# Or query on demand
$ mempalace search "auth" > r.txt
