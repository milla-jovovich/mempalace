# Migration assets

This directory holds the conversion work for folding MemPalace into your own conventions.

## Files

- `mempalace_migration_checklist.md` — sequence for adapting the repo without breaking package, runtime, ingest, hooks, or MCP wiring.
- `mempalace_rebrand_compile_plan.md` — rebrand and compile plan with execution order.

## Companion tooling

See `tools/mempalace_execution_kit/` for the runnable rewrite kit.
