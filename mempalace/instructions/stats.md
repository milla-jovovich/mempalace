# MemPalace Stats

Show a compact summary of the current palace:

- total wings
- total halls
- total rooms
- total memories

Run:

```bash
mempalace stats
```

Optional custom palace path:

```bash
mempalace --palace /path/to/palace stats
```

If no palace exists yet, initialize and mine one first:

```bash
mempalace init <dir>
mempalace mine <dir>
```
